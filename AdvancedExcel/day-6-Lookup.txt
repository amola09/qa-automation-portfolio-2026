Notes:
	Lookup value must be present in first column in table array		
	If Duplicate Lookup value present in table array it gives first occurrence only		
	Works Left to right		
	Column index should be positive		
	Make sure table array reference should be fixed		
			
			
			
LOOKUP	Syntax:		
VLOOKUP	VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])		Exact Match= False / 0
HLOOKUP	HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])		Approximaate= True / 1
XLOOKUP	XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode], [search_mode])		
			
MATCH	MATCH(Lookup value,Lookup array)	Return index of lookup value	
INDEX	INDEX(array, row number)	        Return row number value provided array	
			
Combination of INDEX+MATCH for LOOKUP	INDEX(B1:B16,MATCH(2020,D1:D16))		
	B1:B16 ==> return array		
	2020 ==> lookup value		
	D1:D16 ==> lookup array
